## ✨ Overview

This mod makes clay forming significantly easier by allowing you to:

> 🖱️ **Hold down the right mouse button to automatically complete clay layers**

Instead of manually placing each voxel, the mod modifies the
**"Duplicate Layer"** mode to **auto-fill the current layer**.

## 🎮 How to Use

1. Place clay
2. Select a recipe
3. Press **F** to change mode
4. Select **"Duplicate Layer"**
5. Hold **Right Mouse Button**

---

## 💻 Client-Side Only

This mod:

* Runs entirely on the **client**
* Does **not require server-side installation**
* Should work on most servers without issues

---

## ⚖️ Game Balance

This mod **does NOT break balance**:

* Same clay usage as vanilla
* Same speed as manual forming
* Only reduces repetitive clicking

---

## 🧠 Technical Details

* Uses a **Harmony patch** on:

  * `BlockEntityClayForm.OnCopyLayer`
* Hooks into:

  * `SendUseOverPacket`
* Changes behavior of:

  * `"Duplicate Layer"` mode
